## 数值字符串 => 数字
```js
+ "123"; // 等价于 Number("123")
- "123";  // -123

Number("123");
new Number("123"); //返回数字的包装对象

parseInt("123");
parseFloat("123");

// 转为整数数字
5.2 | 0; // 5
"5.2" | 0; // 5

5.2<<0; // 5
"5.2"<<0; // 5

~~5.2; // 5
~~"5.2"; // 5


```

## 条件语句（if，while...）中非布尔值的转换
```
0、-0、null、""、undefined 和 NaN 都转换为false
```

## 基本类型 => 数字
```js
+ "123";  // 123
+ ""; // 0
+ true; // 1
+ false; // 0
+ null; // 0
+ undefined; // NaN
+ NaN; // NaN
```
